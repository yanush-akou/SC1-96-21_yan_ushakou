Форматы файлов.

.DOC или .doc (от англ document ) - расширение имени файла, используемое для файлов,предствляющих текст,с разметкой или без. Расширение .DOC раньше части использовалось для обозначения простыз текстовых файлов без формирования, однако позже стало использоваться для двоичных форматов с разметкой.
Документ MS Word

Portable document format ( PDF) - межплатформенный формат электронных документов, разработанный фирмой Adobe systems с использованием ряда возможностей языка Postscript.
Документ PDF

ZIP - популярный формат сжатия данных и архивации файлов. Файл в этом формате обычно имеет расширение .zip и хранит в сжатом или несжатом формате один или несколько файлов, которое можно из него извлечь путем распоковки с помощью специальной программы.
